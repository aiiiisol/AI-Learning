# Cursor

## 简介
Cursor 是一款基于 VS Code 开发的下一代 AI 编程编辑器，它将 AI 能力深度集成到了编辑器的每一个角落。

## 核心能力
- **原生 AI 集成**：不同于插件形式，Cursor 在内核层面集成了 AI，支持 Tab 补全、代码库全局对话和 Composer 模式。
- **代码库索引**：能够本地索引整个项目，让 AI 深入理解项目架构和上下文。
- **Composer 模式**：支持一键生成并修改多个文件，极大提升了从零开发新功能的效率。
- **模型切换**：内置了 Claude 3.5 Sonnet, GPT-4o 等顶级模型供用户选择。
