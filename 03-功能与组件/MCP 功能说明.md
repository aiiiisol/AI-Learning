# MCP (Model Context Protocol) 功能说明

MCP 配置完成后，AI 助手获得了直接与本地系统和外部互联网交互的能力，由单一的聊天机器人进化为“自动化智能体”。

## 核心能力列表

### 1. 互联网深度搜索与实时情报 (`brave-search`)
*   **实时查阅文档**：在编写代码时实时搜索最新的库文档、API 规范或解决特定报错。
*   **市场/行业调研**：搜索最新的技术趋势、竞品调研，并汇总成结构化报告。
*   **本地生活查询**：查找周边的服务、餐厅或办公用品供应商信息。

### 2. 全自动文件与代码库管理 (`filesystem`)
*   **代码库维护**：扫描项目文件夹、理解结构、进行重构，并直接将代码写入文件。
*   **自动化笔记整理**：读取 Obsidian 库，整理零散笔记，建立双向链接，或根据对话内容自动生成文档。
*   **数据处理**：批量读取 CSV/JSON 等数据文件，进行清洗、转换或分析。
*   **项目初始化**：一键生成完整的项目脚手架（含配置文件）。

### 3. 浏览器自动化与视觉交互 (`puppeteer`)
*   **网页数据抓取 (Web Scraping)**：模拟点击和滚动，抓取需要登录或 JS 执行后的复杂网页。
*   **前端自动化测试**：启动浏览器，执行功能点击，截图反馈进行视觉和逻辑校验。
*   **静态网页转图片/PDF**：捕获长图或保存网页以便离线查阅。
*   **模拟用户操作**：自动填写表单、下达指令，模拟人类在浏览器上的完整交互。

### 4. 跨平台联动（综合应用场景）
*   **端到端开发**：搜索技术 -> 编写代码 -> 浏览器运行自测。
*   **知识闭环**：搜索热点 -> 抓取详情 -> 整理并存入 Obsidian。
